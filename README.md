# Ty2yAntiVirus
开源杀毒软件

官网：https://www.fairysoftware.com/

专业的、正二八经的杀毒软件。

# 简介
小巧易用、简单轻便的杀毒软件。

病毒查杀、实时防护皆备，即可杀毒，又可防毒。

轻量级，无繁杂的设置、无复杂的使用。

没有多余的进程，没有众多的API监控，对系统性能无影响，不影响电脑运行速度。

还有众多轻便功能，如：开机自启、病毒库自动更新、三种不同扫描模式。
